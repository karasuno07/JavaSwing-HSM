Huong dan cai dat phan mem quan ly ban hang Hotaru (Hotaru Sales Mangement Software)

Yêu cau may tinh su dung he dieu hành Windows (Khuyen nghi Windows 10), JDK 1.8 tro lên
có cài dat SQL Server 2019 tro lên và phan mem Microsoft SQL Server Management Studio
link tai JDK 11: https://www.oracle.com/java/technologies/javase-jdk11-downloads.html
link tai SQLServer: https://www.microsoft.com/en-us/sql-server/sql-server-downloads
link tai MSSMS: https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15

Sau khi dap ung du cau hinh he thong, tien hanh cai dat ung dung Hotaru

- Buoc 1: Mo file database.sql trong thu muc giai nén, chay code khoi tao database bang cách chon tat ca text roi an F5.
- Buoc 2: Doi mat khau user 'sa' thanh 79461382465 bang doan code sau

	ALTER LOGIN sa
	WITH PASSWORD = '79461382465'
	GO

- Buoc 2: Chay file Setup.exe trong thu muc giai nen de cài dat ung dung vao may tinh
- Buoc 3: Mo shortcut Hotaru.exe tren man hinh desktop de khoi dong ung dung, dang nhap vao tai khoan mac dinh: admin, mat khau: 123456
